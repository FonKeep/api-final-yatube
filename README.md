# API для Yatube

## Описание
REST API для Yatube. Проект позволяет пользователям публиковать посты, комментировать их, подписываться на авторов и просматривать тематические группы.

## Технологии
- Python 3.9
- Django 3.2
- Django REST Framework
- Djoser (JWT Authentication)

## Как запустить проект:
1. Клонировать репозиторий:
   git clone https://github.com/FonKeep/api-final-yatube.git
   python manage.py runserver
